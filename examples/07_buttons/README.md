# Example 07 — Buttons

A minimal starting point for exploring UI button interactions in lge.

## What this example shows

- The common example shell (window, input hints, common controls) with no additional entities or logic

## Controls

| Key / Button   | Action                  |
|----------------|-------------------------|
| `F5` / START   | Toggle debug draw       |
| `F11` / SELECT | Toggle fullscreen       |
| `Esc` / B      | Exit                    |
